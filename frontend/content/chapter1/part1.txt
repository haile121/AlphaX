1.1 Generations of programming language. Programming languages are categorized into five generations: (1st, 2nd, 3rd, 4th and 5th generation languages)

 Low level languages are machine specific or dependent. o High level languages like COBOL, BASIC are machine independent and can run on variety of computers. 

➣ Programming languages በአጠቃላይ በሁለት ይከፈላሉ። እነሱም low level እና High level languages ተብለው ነው።

➺ Low level languages are machine specific or dependent. ➺ High level languages like COBOL, BASIC are machine independent and can run on variety of computers. 

Low level የምንላቸው የኮምፒውተር ወይም ማሽን ጥገኛ ናቸው። ይህም ማለት ለአንዱ computer የጻፋችሁት code ለሌላው computer አይሆንም ወይም ሁሉም የየራሳቸው የሚጻፉባቸው computer-ኦች አሏቸው ማለት ነው።

High level የምንላቸው ግን በአንዱ computer ተጽፈው በሌላኛውም መጠቀም የምንችላቸው አይነት languages ናቸው። ለዚያም ነው machine independent ወይም በኮምፒውተር ጥገኛ ያልሆኑ።

➛ From the five categories of programming languages, first and second generation languages are low level languages and the rest are high level programminglanguages. ➛ The higher the level of a language, the easier it is to understand and use by programmers. ➛ Languages after the fourth generation are referred to as a very high level language. 

➣ ከprogramming language Generation ውስጥ የመጀመሪያወቹ (1st Generation) እና ሁሀተኛወቹ (2nd Generation) Low level Language ውስጥ ሲመደቡ ሌላዎቹ(3rd, 4th, 5th) ደግሞ High level ናቸው።

➣ ቋንቋው High level እየሆነ ሲመጣ ለprogrammer-ኦች ወይም ለእናንተ ለመጠቀም እየቀለለ ይመጣል።

➣ ከ4th Generation በኋላ ያሉት programming languages Very High Level Programming languages ይባላሉ።

እስኪ ሁሉንም Generation-ኦች አንድ በአንድ እንያቸው። እዚህ ላይ ግን ሁሉንም ነገር በቃላችሁ ለመያዝ አትሞክሩ በአጭሩ አንድ 3 point ከዪዛችሁ ይበቃል ሌላውን concept-ን በመረዳት መመለስ ትችላላችሁ።

1.1.1. First Generation (Machine languages, 1940’s): 

የመጀመሪያወቹ ትውልዶች😊(1st Generation) በሌላ ስማቸው Machine Languages ይባላሉ።Difficult to write applications with. 

እነዚህን ቋንቋዎች ተጠቅመን ተግባሮችን ለመጻፍ አስቸጋሪ ናቸው።Dependent on machine languages of the specific computer being used. 

ከላይ እንዳልነው low level ስለሆኑ ለኮምፒዩተሩ ወይም ለmachine-ኡ ጥገኛ ናቸው።Is more powerful in utilizing resources of the computer. 

የኮምፒውተሩን አቅም አሟጠው ነው የሚጠቀሙት። አንበጣ በሏቸው😊

1.1.2. Second Generation (Assembly languages, early 1950’s): 

ሁለተኛዎቹ ትውልዶች Assembly Languages ይባላሉ። ይሄውም Assembler የሚባለውን ስለሚጠቀሙ ነው። (ምንድን ነው ደግሞ እሱ እንዳትሉ ወደ ታች በሰፊው እናየዋለን😊)Uses symbolic names for operations and storage locations. 

እነዚህ ቋንቋዎች የውክልና ስሞችን ለሚሰሩ ስራዎች እና ማህደር ላይ ለማስቀመጥ ይጠቀማሉ።A system program called an assembler translates a program written in assembly language to machine language. 

ይሄ ፈተና ላይ በተደጋጋሚ ይመጣል። Assembler ማለት በአይን የማይታይ ግን በcomputer program ውስጥ ያለ Software ሲሆን assembly language-ን ወደ Machine Language ይቀይርልናል። Programs written in assembly language are not portable. i.e., different computer architectures have their own machine and assembly languages. They are highly used in system software development. 

በassembly language የተጻፉ programm-ኦች ከቦታ ቦታ ወይም ከአንድ computer ወደ ሌላ computer ወስደን መጠቀም አይቻልም። ምክንያቱም እነዚህም low level language ስለሆኑ ነው።

1.1.3. Third Generation (High level languages, 1950’s to 1970’s): 

እስካሁን ያሉት Generation-ኦች (1st, 2nd) low level ሲሆኑ ከዚህ ጀምሮ ያሉት ደግሞ high level languages ይባላሉ።Uses English like instructions and mathematicians were able to define variables with statements such as Z = A + B 

እንግሊዝኛና maths የሚመስሉ ቋንቋዎችን የቀላቀሉ ናቸው። እንግሊዝኛው part ኮምፒውተሩን የምናዝበት ነው። ለምሳሌ Add ብንል ደምርልን እያልነው ነው።Such languages are much easier to use than assembly language. 

ከሁለተኛው Generation (assembly language) ቀለል ይላሉ። ምክንያቱም የlanguage-ኦች generation እያደገ ሲሄድ ቋንቋዎች ይቀላሉ ብለናልና።Programs written in high level languages need to be translated into machine language in order to be executed. 

ወደ ስራ ለመዋል ወደ machine language (1st generation) መቀየር አለባቸው።

The use of common words (reserved words) within instructions makes them easier to learn. 

እነዚህን እንግሊዘኛ ቃላት መጠቀሙ የበለጠ ለመማር ቀላል ያደርጋቸዋል።All third generation programming languages are procedural languages. 

ሁሉም 3rd generation ቋንቋዎች procedural languages ናቸው። ስለዚህ what & how ላይ ትኩረት ያደርጋሉ። (ደግሞ procedural ምንድን ነው እንዳትሉ ወደ ታች በሰፊው እናየዋለን😊) 

Have a simple, English like syntax rules; commonly used to access databases. 

አጫጭር ቀላል እንግሊዝኛ ቃላቶችን ይጠቀማል። በተለይ database (የመረጃ ማጠራቀሚያ) ከሱ ጋር በተያያዘ ኮምፒውተራችን ስናዘው የምንጠቀማቸው ቋንቋዎች ናቸው።

1.1.4. Fourth Generation (since late 1970’s): 

Forth generation languages are non-procedural languages. The non-procedural method is easier to write

እነዚህ ከ3rd Generation የሚለዩበት ሌላው non-procedural መሆናቸው ነው። ይሄውም ለመጻፍ ቀላል ሲሆኑ የበለጠ የሚያተኩሩት what ላይ ነው። የ3rd generation-ኦች ግን what እና how ላይ እንደነበር ታስታውሳላችሁ አይደል?።

Forth generation languages have a minimum number of syntax rules. 

code-ኦቹን የምንጽፍበት ህግ (syntax) በጣም ትንሽ ወይም ቀላል ናቸው።

የ 4GL(4th Generation Languages) ምሳሌዎች structured query languages (SQL), report generators, application generators and graphics languages የመሳሰሉት ናቸው።

1.1.5. Fifth Generation (1990’s): 

These are used in artificial intelligence (AI) and expert systems; also used for accessing databases. 

እነዚህ የመጨረሻዎቹ ደረጃ ትውልዶች ሲሆኑ (ሌላ እስከሚፈጠር ድረስ ማለት ነው😊)  AI ወይም Artificial Intelligence  የሚባለው ጨምሮ አሁን አለማችን ላይ ትላልቅ ስራዎች የሚሰሩባቸው የprogramming language  አይነቶች ናቸው። እንደ 4th Generation-ኦች database-ን ለማንበብ እና ለመጠቀምም ያገለግላሉ።5GLs are “natural” languages whose instruction closely resembles human speech. E.g. “get me Jone Brown’s sales figure for the 1997 financial year”. 

እነዚህ ቋንቋዎች እኛ የምንጠቀማቸው አይነት ቋንቋዎች ናቸው። ለምሳሌ ChatGpt ወይም DeepSeek ወይም ሌላ AI እየተጠቀማችሁ Explain about C++ ብትሉት በቀላሉ ይመልስላችኋል አይደል። ይሄ የሆነው የቱ?😄 "እናንተና ኮምፒውተሩ በቀላሉ ልትግባቡ የቻላችሁ" programming language-ኦች 5th Generation ስለደረሱ ነው። 

መጽሐፋችሁ ላይ ያለው ምሳሌም Jhon የሚባለውን ሰውየ በ1997 የሸጠውን አጠቃላይ ዝርዝር እንዲሰጠው እየጠየቀው ነው። ይሄውም ቀጥታ እንግሊዝኛ ተጠቅሞ ነው። 1at generation ቢሆን ግን 0 እና 1 -ን ብቻ ነበር የሚረዳው ስለዚያ 11110000 እየተባለ ይጻፋል ማለት ነው።5GLs require very powerful hardware and software because of the complexity involved in interpreting commands in human language. 

እነዚህ ቋንቋዎች ለመጠቀም ኮምፒውተራችሁ በጣም አስተማማኝ የሆነ የhardware (የኮምፒውተሩ ውጫዊ ክፍል፣ ልንነካው ልንዳስሰው የምንችለው part) እና ነsoftware (የኮምፒውተሩ ውስጣዊ የማይታየው ክፍል) አካላት ሊኖሩት ይገባል።







1.2. Overview of Computers and Computer Organization. 

ስለ ኮምፒውተር ምንነት ወይም ኮምፒውተር በውስጡ የሚይዛቸውን ነገሮች የሚያጠና topic ነው። 

Regardless of differences in physical appearance, virtually every computer may be envisioned as being divided into six logical units or sections: 

ኮምፒውተሮች በአካል ስናያቸው የተለያየ ቅርጽ ነው ያላቸው። ለምሳሌ dell computer እና asus የተለያየ ቅርጽ አላቸው። ነገር ግን እንዳጠቃላይ ማንኛውም computer እነዚህ 6 ክፍሎች ይኖሩታል።➊ Input Unit: it obtains information from various input devices and places this information at the disposal of the other units so that the information may be processed or stored. 

መረጃ ወደ computer-ሩ የምናስገባበት ነው። መረጃ ስንል ድምጽ ሊሆን ይችላል፣ ጽሑፍ የመሳሰሉትን።  ከinput unit መካከል እንደ keyboard⌨️, camera📷, የመሳሰሉትን መጥቀስ እንችላለን።

➋ Output Unit: it takes information that has been processed by the computer and places it on various output devices to make the information available for use outside the computer. 

ይሄ ደግሞ በተቃራኒው መረጃ ከኮምፒውተሩ ወደ እኛ የሚመጣባቸው ክፍሎች ናቸው። ለምሳሌ monitor💻(የኮምፒውተሩ screen ማለት ነው።), speaker የመሳሰሉትን መጥቀስ ይቻላል።
